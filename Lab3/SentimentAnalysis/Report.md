# Звіт про виконання Лабораторної роботи №3

**Тема:** Аналіз тональності коментарів веб-сайтів за допомогою бінарної класифікації в ML.NET.

## 1. Мета роботи
Створити консольний додаток на C# (платформа .NET 9.0) для класифікації тональності відгуків (позитивний або негативний) з використанням бібліотеки машинного навчання ML.NET. 

## 2. Виконані кроки

### 2.1. Створення та налаштування проекту
1. Створено консольний додаток `SentimentAnalysis`.
2. Встановлено NuGet-пакет `Microsoft.ML` (версія 5.0.0).

### 2.2. Підготовка даних
1. Використано набір даних **UCI Sentiment Labeled Sentences dataset**.
2. Файл `yelp_labelled.txt` (відгуки Yelp) скопійовано у внутрішню папку `Data`.
3. Файл налаштовано на автоматичне копіювання до вихідного каталогу під час збірки (`Copy if newer` / `PreserveNewest`).

### 2.3. Програмна реалізація
Було розроблено дві основні сутності даних у файлі `SentimentData.cs`:
- `SentimentData` — містить текст коментаря (`SentimentText`) та його мітку (`Sentiment`).
- `SentimentPrediction` — наслідує `SentimentData` та додає поля для результатів моделі: передбачена мітка (`Prediction`), ймовірність (`Probability`) та сира оцінка (`Score`).

**Алгоритм роботи в `Program.cs`:**
1. **Ініціалізація MLContext:** Створено середовище ML.NET, спільне для всіх об'єктів робочого процесу.
2. **Завантаження даних (`LoadData`):** Дані з текстового файлу завантажуються в об'єкт `IDataView`.
3. **Розподіл даних:** Набір даних поділено на тренувальний (80%) та тестовий (20%) за допомогою методу `TrainTestSplit`.
4. **Конвеєр (Pipeline) та тренування (`BuildAndTrainModel`):** 
   - Текст конвертується у числові вектори (`FeaturizeText`).
   - Додається алгоритм бінарної класифікації **SdcaLogisticRegression** (Стохастичний подвійний координатний підйом).
   - Модель тренується викликом `Fit()`.
5. **Оцінка моделі (`Evaluate`):** Трансформовано тестовий набір даних і розраховано показники якості за допомогою `CalibratedBinaryClassificationMetrics`.

## 3. Результати тестування
Під час перевірки на тестовій вибірці модель показала такі результати ефективності:

* **Accuracy (Точність):** 83.96% (відсоток правильних прогнозів).
* **AUC (Площа під ROC-кривою):** 90.06% (впевненість моделі при класифікації позитивних/негативних класів).
* **F1Score (F1-міра):** 84.38% (баланс між точністю та повнотою).

### Тестування прогнозування (Prediction)
Модель було успішно перевірено на нових, раніше небачених даних:

**Тестування одиночного запису:**
- *Вхідний текст:* "This was a very bad steak"
- *Прогноз:* Negative (Ймовірність: ~0.028)

**Тестування групи записів (Batch):**
- *Вхідний текст:* "This was a horrible meal"
  *Прогноз:* Negative (Ймовірність: ~0.041)
- *Вхідний текст:* "I love this spaghetti."
  *Прогноз:* Positive (Ймовірність: ~0.997)

## 4. Висновок
 Побудована модель бінарної класифікації в ML.NET здатна коректно розпізнавати позитивну та негативну тональність англомовних коментарів.
